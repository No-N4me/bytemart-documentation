# 📦 ArkKits (Example Plugin)

ArkKits is an advanced plugin for Ark: Survival Evolved / Ascended that allows server admins to seamlessly distribute customizable starter items, dinos, and buffs to players based on permissions.

## Features
- **Dynamic Cooldowns:** Prevent abuse with strict database-enforced cooldowns.
- **Dino Spawning:** Spawn leveled, saddled tames directly to players.
- **Discord Integration:** Log kit claims directly to a Discord webhook.

## Installation
1. Ensure you have the latest version of [ArkApi](https://arkserverapi.com/) installed on your server.
2. Download the `.zip` file from your **Bytemarts Dashboard**.
3. Extract the `ArkKits` folder.
4. Drop the folder into `ShooterGame/Binaries/Win64/ArkApi/Plugins`.
5. Restart or launch your Ark server.
